    Tools → EveryWear → Account Status
    → ログイン状態確認
    → 有効なサブスクリプション確認

    **2. アバターファイル形式確認**
    ```
    サポート形式: FBX, OBJ
    推奨: VRChat標準のFBX形式
    ボーン構造: Humanoid互換必須
    ```

    **3. システム要件再確認**
    ```
    インターネット接続: 必須
    ファイアウォール: EveryWear通信許可
    ウィルスソフト: 一時無効化テスト
    ```
